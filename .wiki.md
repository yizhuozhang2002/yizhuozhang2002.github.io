# Project Summary
The project is a personal portfolio website for Yizhuo Zhang, a Frontend/Full-Stack Engineer and Master of Computing candidate at NUS. It showcases expertise in developing high-performance dashboards, intelligent user interfaces, and AI-driven applications, emphasizing design and user experience.

# Project Module Description
The portfolio consists of several sections:
- **Hero**: Engaging introduction with animated background and typing effect.
- **About Me**: Overview of Yizhuo's background and skills, including a profile photo.
- **Education**: Timeline of educational achievements.
- **Skills**: Display of technical skills with animated progress bars.
- **Experience & Research**: Detailed timeline of projects and research contributions.
- **Projects**: Highlighting key projects with descriptions and clickable links.
- **Publications**: List of academic publications with access hints.
- **Honours & Activities**: Recognition and extracurricular activities.
- **Contact Form**: For visitors to reach out.

# File Description Inventory
- **images/**: Holds image files including backgrounds and resumes.
- **index.html**: The main structure of the portfolio website.
- **package.json**: Lists project dependencies and scripts for building and running the project.
- **script.js**: Contains JavaScript functions for animations and interactivity.
- **style.css**: Defines the visual style of the portfolio.
- **template_config.json**: Configuration settings for templates used in the project.

# Technology Stack
- HTML
- CSS
- JavaScript
- Libraries: React, Bootstrap, D3.js, and others for animations and design.
- Tools: Git for version control, npm for package management.

# Usage
1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the project:
   ```bash
   npm run build
   ```
3. Run the project locally:
   ```bash
   npm start
   ```
