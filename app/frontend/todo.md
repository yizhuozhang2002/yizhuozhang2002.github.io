# Yizhuo Zhang - Personal Portfolio Website Redesign

## Design Guidelines

### Design References
- **Apple.com**: Clean minimalism, elegant typography, smooth animations
- **Linear.app**: Dark mode, glassmorphism, gradient accents
- **Stripe.com**: Dynamic gradient backgrounds, micro-interactions

### Color Palette
- Primary Background: #0a0a0f (Deep Space Black)
- Secondary Background: #12121a (Dark Navy)
- Card Background: rgba(255, 255, 255, 0.05) (Glassmorphism)
- Accent Primary: #6366f1 (Indigo)
- Accent Secondary: #8b5cf6 (Purple)
- Accent Gradient: linear-gradient(135deg, #6366f1, #8b5cf6, #ec4899)
- Text Primary: #f0f0f5 (Off White)
- Text Secondary: #9ca3af (Cool Gray)
- Border: rgba(255, 255, 255, 0.1)

### Typography
- Headings: 'Inter', sans-serif (700 weight)
- Body: 'Inter', sans-serif (400 weight)
- Mono/Code: 'JetBrains Mono', monospace

### Key Component Styles
- Cards: Glassmorphism with backdrop-blur, subtle border glow on hover
- Buttons: Gradient background with glow effect on hover
- Timeline: Animated line with glowing dots
- Skill bars: Animated gradient fill on scroll

### Animations
- Particles.js background in hero
- Typing effect for hero subtitle
- Scroll-triggered fade-in/slide-up animations via Intersection Observer
- 3D tilt effect on project cards
- Smooth parallax scrolling
- Gradient border animation on hover

## Development Tasks

1. **index.html** - Complete HTML structure with all sections
2. **style.css** - Full modern dark theme styling with animations
3. **script.js** - All interactive features (particles, typing, scroll animations, tilt, nav)
